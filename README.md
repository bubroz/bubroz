## The Digital Intelligence Ecosystem

A suite of specialized tools designed for raw data ingestion, semantic enrichment, and persistent archival.

| Project | Description |
|---------|-------------|
| **[zget](https://github.com/bubroz/zget)** | **The Archival Engine**. High-fidelity media system. Features **The Archivist** (Ingestion), **The Vault** (Persistence), and **The Portal** (PWA). |
| **[Waypoint](https://github.com/bubroz/waypoint)** | **Location Intelligence API**. Transforms raw coordinates into semantic context and AI-powered narratives. Integrating **Gemini 3 Flash** and **MCP**. |
| **[Groundwork](https://github.com/bubroz/groundwork)** | **Property Forensics Engine**. Automated high-fidelity extraction from 3,143 counties. Operates as an **MCP Server** ("The Hands"). |
| **Librarian** | **Multimodal Intelligence**. Extracts structured, evidence-backed insights from audio/video using SAM3 and ECAPA-TDNN Voiceprints. |
| **Chimera** | **Agentic Analytics**. Advanced reasoning platform (ACH, Pre-Mortem, Red Hat) using LLMs + high-fidelity RAG. |
